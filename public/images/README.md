# Images Directory

This directory should contain images for the YDO website.

## Recommended Images

### Homepage
- `hero-youth.jpg` - Hero image for the homepage (suggested: youth/community members in Northern Uganda)

### About Page
- Leadership photos of Simon Peter Wilobo
- Community photos showing YDO's work

### Programs Page
- Education: Children in school, students learning
- Vocational Training: Training sessions, skills development
- Psycho-Social Support: Support groups, counseling activities
- Advocacy: Community meetings, child welfare activities
- Economic Empowerment: Women's groups, businesses, cooperatives

### Impact Page
- Community transformation photos
- Before/after shots showing impact
- Success stories

### Partnership Page
- Photos of partnership activities with Be A Number
- Collaborative work sessions

## Image Guidelines

- Use high-quality images that reflect YDO's work
- Ensure images are properly optimized for web (use Next.js Image component)
- Respect privacy and obtain proper consent for photos of individuals
- Images should align with YDO's values of dignity, community ownership, and empowerment
